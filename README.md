# Claude's Secret Thinking Words

> Claude Code 思考时会显示的 184 个隐藏词汇，全部挖出来做成了交互式探索器。

**[Live Demo](https://pipiquan352.github.io/claude-thinking-words/)** | **[Interactive Course](https://pipiquan352.github.io/claude-thinking-words/course.html)**

## What is this?

When Claude Code is thinking, it shows a spinner with random words like *Simmering*, *Moonwalking*, or *Discombobulating*. These 176 spinner words (+ 8 completion words) are hardcoded in `cli.js` — someone extracted them all and built this explorer.

## Features

- **9 categories** — Serious Thinking, Cooking, Nature/Science, Silly Actions, Nonsense, Magic/Fantasy, and more
- **Bilingual** — Every word has an emoji and Chinese translation
- **Slot machine** — Click "Spin" to get a random word with a satisfying deceleration animation
- **Search** — Full-text search in both English and Chinese
- **Collection game** — Track how many words you've discovered, with milestone rewards every 10 words
- **Easter eggs** — 8 hidden completion words to find

## The Course

A 6-module interactive course that teaches how the explorer's code works — from data structures to CSS animations — with quizzes, group chat visualizations, and code-to-plain-language translations.

## Tech

Single self-contained HTML file. No frameworks, no build tools, no dependencies. Just open and play.
